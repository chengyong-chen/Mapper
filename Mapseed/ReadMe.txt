========================================================================
    ACTIVEX 控件 DLL : Mapseed 项目概况
========================================================================

控件向导为您的 Mapseed ActiveX 控件 
DLL 创建了此项目，其中包含一个控件。

此主干项目不仅说明了编写 
ActiveX 控件的基础知识，而且是编写特定控件
功能的起点。

此文件包含组成 Mapseed ActiveX 控件 DLL
的每个文件的内容摘要。

Mapseed.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主
    项目文件。它包含有关生成该文件的
    Visual C++ 版本的信息，以及有关使用应用程序向导选择的平台、
    配置和项目功能的信息。

Mapseed.h
    这是 ActiveX 控件 DLL 的主包含文件。
    它包括其他项目特定的包含文件，如 resource.h。

Mapseed.cpp
    这是包含 DLL 初始化、终止和其他
    簿记的代码的主源文件。

Mapseed.rc
    这是项目使用的 Microsoft Windows 资源列表。
    可使用 Visual C++ 资源编辑器
    直接编辑此文件。

Mapseed.def
    此文件包含有关运行 Microsoft Windows 
    所需的 ActiveX 控件 DLL 的信息。

Mapseed.idl
    此文件包含控件
    类型库的对象描述语言源代码。

Mapseed.ico
    此文件包含将在“关于”框中显示的图标。此图标
    位于主资源文件 Mapseed.rc 中。

/////////////////////////////////////////////////////////////////////////////
CMapseedCtrl 控件:

MapseedCtrl.h
    此文件包含 CMapseedCtrl C++ 类的声明。

MapseedCtrl.cpp
    此文件包含 CMapseedCtrl C++ 类的实现。

MapseedPropPage.h
    此文件包含 CMapseedPropPage C++ 类的声明。

MapseedPropPage.cpp
    此文件包含 CMapseedPropPage C++ 类的实现。

CMapseedCtrl.bmp
    此文件包含一个位图，容器将在
    CMapseedCtrl 控件出现在工具面板上时使用它来表示该控件。此位图
    位于主资源文件 Mapseed.rc 中。

/////////////////////////////////////////////////////////////////////////////
帮助支持:

Mapseed.hpj
    此文件是帮助项目文件，由帮助编译器用来创建
    ActiveX 控件的帮助文件。

*.bmp
    这些是 Microsoft 基础类库标准
    命令的标准帮助文件主题所需的
    位图文件。这些文件位于 HLP 子目录下。

Mapseed.rtf
    此文件包含许多 ActiveX 控件支持的
    通用属性、事件和方法的标准帮助主题。
    您可以编辑此文件以添加或移除其他控件特定主题。
    此文件位于 HLP 子目录下。

/////////////////////////////////////////////////////////////////////////////
授权支持:

Mapseed.lic
    这是用户许可证文件。此文件必须同控件的 DLL 出现在相同的目录中，
    以便在设计时环境中创建控件的一个实例。
    您通常会将此文件同控件一同分发，
    但您的客户不会将其分发。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

stdafx.h、stdafx.cpp
    这些文件用于生成名为 Mapseed.pch
    的预编译头(PCH)文件以及名为 stdafx.obj 的预编译类型(PCT)文件。

resource.h
    这是标准的头文件，它定义了新的资源 ID。
    Visual C++ 资源编辑器读取和更新此文件。

/////////////////////////////////////////////////////////////////////////////
其他注释:

控件向导使用 "TODO:" 指示
应增加或自定义的源代码部分。
